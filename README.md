# AI Code Review Assistant

A web app that reviews your code and gives you instant feedback — quality score, security warnings, bug detection, and improvement suggestions, all generated by Groq AI.

Paste your code, pick a language, and get a structured breakdown in seconds.

**Live:** [devreview-ai.vercel.app](https://devreview-ai.vercel.app)

---

## What it does

You paste code into the editor, select the language, and the app sends it to Groq AI through a secure API route. The AI returns a structured analysis — not just a wall of text, but scored categories you can actually act on.

The feedback covers:

- **Overall score** out of 100
- **Code quality** — readability, naming, structure
- **Security warnings** — unsanitised inputs, exposed secrets, common vulnerabilities
- **Bug detection** — logic errors, edge cases, potential crashes
- **Performance** — inefficient patterns, unnecessary computations
- **Recommendations** — specific things to change, with reasoning

## Getting started

**Prerequisites:** Node.js 18+, a [Groq API key](https://console.groq.com)

```bash
# Clone the repo
git clone https://github.com/carinadesouza/AI-Code-Review-Assistant.git
cd AI-Code-Review-Assistant

# Install dependencies
npm install

# Add your Groq API key
echo "GROQ_API_KEY=your_key_here" > .env.local

# Run locally
npm run dev
```

App runs at `http://localhost:3000`

## How the AI response is structured

The API route instructs Groq to return JSON in this format:

```json
{
  "overallScore": 85,
  "summary": "Readable code with minor security and performance concerns.",
  "codeQuality": ["Functions are modular", "Naming is consistent"],
  "securityWarnings": ["User input is not sanitised"],
  "improvements": ["Add input validation", "Reduce repeated computations"],
  "recommendations": ["Add error handling", "Use consistent formatting"]
}
```

The frontend parses this and renders each section as its own card — so the output is scannable, not a block of prose.
