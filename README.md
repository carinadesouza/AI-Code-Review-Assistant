# AI Code Reviewer

AI Code Reviewer is a modern full-stack web application that allows developers to paste code and receive an AI-powered analysis instantly. The system evaluates code quality, detects security risks, suggests improvements, and provides best-practice recommendations.

The application is designed with a **modern developer-tool interface** and powered by **Groq AI** for fast analysis.

---

# Overview

AI Code Reviewer acts as an intelligent assistant for developers. Users paste code into the editor, select a programming language, and receive structured feedback generated by an AI model.

The analysis includes:

- Code quality evaluation
- Security vulnerability detection
- Bug identification
- Maintainability insights
- Performance concerns
- Best-practice recommendations

The UI follows a **clean SaaS-style developer aesthetic** with a polished and responsive interface.

# How It Works

1. The user pastes code into the editor.
2. The user selects the programming language.
3. The request is sent to a secure **Next.js API route**.
4. The API route calls the **Groq AI API**.
5. The AI analyzes the code.
6. The AI returns structured JSON.
7. The frontend parses the response and displays results.

---

# AI Response Format

The AI is instructed to return structured JSON:

```json
{
  "overallScore": 85,
  "summary": "The code is readable but contains minor performance and security concerns.",
  "codeQuality": [
    "Functions are modular",
    "Naming conventions are clear"
  ],
  "securityWarnings": [
    "User input is not sanitized"
  ],
  "improvements": [
    "Add input validation",
    "Reduce repeated computations"
  ],
  "recommendations": [
    "Implement error handling",
    "Follow consistent code formatting"
  ]
}

> This format allows the UI to render the analysis cleanly.
